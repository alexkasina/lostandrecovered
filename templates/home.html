Lost and Report web application is a web tool use to find a lost item or report a found item.

Find A Lost Item

If you have lost an item and you wish to use this system to search it, you can either

1. Search using an identification number of the item on the red Button on the left side for the items with identification number e.g identification cards, visa cards etc

 

2. Search with the red Button on the left if the item doesnt have an identification number e.g phone, calculator etc.

  

Report A Found Item

  

Report A Found Item--TEST

To report a found item, use the green icon at the center and fill detailed information so that the item will be easier to retrieve. The details will be sent to the owner incase they searched for the item.
